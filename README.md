# Mini Shop Task

A comprehensive full-stack e-commerce solution featuring a robust backend API, an intuitive web dashboard for management, and a modern mobile application for users.

## Project Structure

This repository is organized into three main sub-projects. Each directory contains its own detailed `README.md` file with specific installation and setup instructions:

- **[Backend](./backend)**: The core API built with NestJS and PostgreSQL.
- **[Dashboard](./dashboard)**: A Vite-powered admin interface for managing products, orders, and users.
- **[Mobile](./mobile)**: A React Native (Expo) application for customer shopping experiences.
